ProStream is a high-performance Unity DOTS/ECS world streaming toolkit focused on turning traditional GameObject scenes into Entity SubScenes with practical editor workflows.

It combines scene setup automation, rule-based conversion, and runtime streaming systems so large worlds are easier to prepare, inspect, and stream efficiently.

The tooling is built for iterative production workflows and includes diagnostics that help catch scene configuration and conversion issues earlier.
